SkyPulse
🌦️ Weather Forecasting Web Application

A modern Weather Forecasting Web Application built using HTML, CSS, and Vanilla JavaScript, designed to demonstrate strong JavaScript fundamentals and real-world API integration. The application allows users to search for weather conditions in any city and view detailed forecasts with a responsive interface optimized for both mobile and desktop devices.

This project focuses heavily on JavaScript architecture, asynchronous programming, and DOM manipulation, making it an ideal learning project for understanding how frontend applications interact with external APIs.

🚀 Features

🔍 Search weather by city name

📍 Detect weather using user's current location

🌡️ Display current temperature and weather conditions

💧 Show humidity, wind speed, and pressure

📅 5-day weather forecast

🔁 Temperature unit toggle (°C / °F)

🌙 Dark / Light mode

🕘 Search history stored with LocalStorage

🎨 Dynamic backgrounds based on weather conditions

📱 Fully responsive design for mobile and laptop

⏳ Loading indicators while fetching data

🧠 JavaScript Concepts Used

This project emphasizes clean, modular, and well-structured JavaScript code. Each function is commented to explain why the logic exists, not just what it does.

Key concepts implemented:

Async/Await & Promise.all for handling parallel API requests efficiently

Fetch API with URL building and HTTP status validation

Robust error handling using try / catch / finally

Geolocation API for retrieving the user's current location

LocalStorage for saving user preferences such as:

-search history

-temperature unit

-theme mode

Dynamic DOM manipulation using:

-createElement

-innerHTML

-classList

Modular JavaScript functions following the single-responsibility principle

CSS variable-based theming controlled through JavaScript for theme switching

🛠️ Tech Stack

-Weather API – Real-time weather data
